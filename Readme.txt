程序运行说明
1.运行环境：
    应在配置有gcc编译器的windows系统中运行；

2.如何编译和运行程序：
    在cmd中输入： g++ Game2048.cpp fun.cpp -o “文件名称” 以生成名称为“文件名称”的.exe文件，
    可选命令行参数： -t （测试模式，即最大值为64） -s “x”（x 为棋盘的维度，取值是2、3、4、5，若不在此范围内则会提示重新设定维度）
    每输入一个其他未定义的参数，程序就会提示“You entered an invalid parameter!”，若未设定维度，则默认是4；
 
    例如:  在cmd中 先输入g++ Game2048.cpp fun.cpp -o Game，生成Game.exe
            然后在中该文件目录下输入 game.exe -t -s 2
            进入棋盘为2×2方格的2048游戏的测试模式。
